# Instrucciones para subir los cambios a github

## La primera vez que lo hacemos:

- git init
- <creamos el fichero>
- git add .
- git commit -m "mensajito"
- git remote add origin git https://github.com/**Manuchei**/misDiagramas.git
- git push origin master
